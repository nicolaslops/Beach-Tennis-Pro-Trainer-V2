# Beach Tennis Pro Trainer - v61

Versao corrigida com visual mais limpo, cores legiveis e imagens locais nos exercicios.

## Conteudo

- 120 exercicios
- 50 planos de aula
- 3 planos de evolucao
- 120 imagens SVG locais para os exercicios

- PWA com manifest e service worker em cache v61

## Importante

Use esta versao v61 no lugar da v60. Se o navegador ainda mostrar a versao antiga, limpe o cache do site ou abra em janela anonima para testar.
